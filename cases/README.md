# knowledge/cases — Layer 6 匿名案例

本資料夾存放**去識別化之實務案例**，供教學與檢索參考。案例僅「引用」規則與法規，不重述規則內容。

## 原則

- **去識別化**：不得包含任何可識別個人之資料。
- **Evidence First / Reference Only**：案例之分析與結論須以 `rule_reference`／`law_reference` 指向 Rule 層與 Regulations／Interpretations，不得在案例內自行建立新規則或 AI 推論。
- **Manual Review**：結論不確定、或屬地方政府個別認定者，`manual_review = true`。

## 檔案格式

每個案例為一個符合 [`case-schema.json`](case-schema.json) 的 JSON 檔。`priority` 建議 70。

## 現況

本版（V1.5.0）僅建立 Schema，尚未建立任何案例。
