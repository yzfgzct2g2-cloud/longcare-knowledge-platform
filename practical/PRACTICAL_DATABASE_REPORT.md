# 長照實務主題知識庫建置報告（PRACTICAL DATABASE REPORT）

第二層資料庫位置：`knowledge/practical/`
建置基準：以第一層法規資料庫 `knowledge/regulations/`（115/07/01 版）為唯一來源。
性質：**實務主題知識庫**（非法規整理）。每個主題之 `summary` 均依現有法規資料庫產生，標示 `source_type: "practical_summary"`，並以 `related_articles` / `related_codes` / `related_appendices` 及 `law_source_note` 保留對應法規來源。

---

## 一、檔案結構

```
knowledge/practical/
├── topics/                       # 20 個主題 JSON
│   ├── topic-001-外籍看護.json
│   ├── topic-002-失智症.json
│   ├── … (至 topic-020)
├── synonym-map.json              # 同義詞 → 標準主題名稱
├── common-question-map.json      # 常見問題 → 相關主題（及條文）
└── PRACTICAL_DATABASE_REPORT.md  # 本報告
```

每個 topic 採以下結構（在指定 schema 上額外保留來源欄位）：
`topic / source_type / aliases / summary / common_questions / related_articles / related_codes / related_appendices / related_topics / keywords / law_source_note`（`身障輔具` 另含 `needs_manual_supplement`）。

## 五、需要人工補充的主題

以下主題之內容部分超出現有法規資料庫範圍，已避免編造、僅就資料庫內可考者撰寫，建議後續人工補充：

1. **身障輔具（topic-016，已標記 `needs_manual_supplement: true`）**
   - 本辦法僅規範「準用身心障礙者輔具費用補助辦法」（第20條）與「相同輔具未達最低使用年限不得重複申請」（第10條第3項）。
   - 「身心障礙者輔具費用補助辦法」之輔具項目、補助金額、補助對象等屬**另一部法規**，不在本長照資料庫內，需另行匯入該辦法後建立對照。

2. **居家無障礙改善（topic-014）— F碼新舊對照**
   - 111版 FA01–FA21 與 115版 FA08/FA13/FA15/FA17–FA19/FA22–FA44 之對應、各碼給付上限與修繕標的文字均有調整。
   - 此為法規資料庫 `FINAL_DATABASE_REPORT.md` 列示之待人工確認事項，建議建立逐項新舊碼對照表後回填本主題。

3. **輔具（topic-013）— E碼規格細節**
   - 已連結全部 47 個 E碼，惟 `appendix-04.json` 之 E碼採結構化欄位、未逐字收錄冗長規格／功能規範；查詢細部規格仍須對照母檔或定稿附表四。
   - 若未來要細分「居家用／社區用」「購置／租賃」子主題，建議再依附表四細目補充。

4. **營養照護（topic-010）／CB01–CB01a 銜接**
   - CB01（111版，115年停用）與 CB01a（新增）之歷史申報停用與銜接時點，依資料庫待人工確認事項，須再確認。

5. **CMS等級（topic-012）用語**
   - 「CMS」為實務通稱，法規明文用語為「長照需要等級（第一級至第八級）」；已於 `law_source_note` 標註。若需對應 CMS 評估量表細項（非本辦法內容），需另行補充。

6. **交通接送（topic-004）— 各地支付價格**
   - 支付價格由各地方主管機關訂定公告（第19條），屬地方公告事項，非本辦法明定，無法由資料庫產生具體金額，需依各縣市公告補充。
